jQuery UI Multiselect

is my attempt of a sortable multiple select widget. It depends on jQuery 1.3 and jQuery UI 1.7. The widget is styleable using Themeroller. It works in an unobtrusive fashion, by just turning html multiple select inputs into a sexier equivalent. There's no extra markup needed.

The source code is available at Github.
